Initial Setup

You will need to create an individual file for each student who is on the radar. The idea is to generate weekly sheets that will be used for data collection. This spreadsheet will automatically create the weekly sheets for you and link each sheet's data to the Summary sheet.

Below are the steps for the initial setup of a student's Radar Data Collection Spreadsheet.

  1. Use this downloaded spreadsheet as a template for each of your students on the radar. Do this by making a copy of the spreadsheet in your Google Drive and placing it in a folder where you can easily access it (for example, as a hyperlink on your Portal or a bookmark in your Chrome browser).
  2. Within the spreadsheet, locate the sheet named Week_Template and enter the student‚Äôs classes in the yellow-colored cells (O2 through X2). The template allows space for up to 10 classes. If you do not need all 10, simply use the first few cells and leave the remaining ones blank. The class information will automatically transfer to the cells in column A of the Week_Template.
  3. Create the weekly data collection sheets by either (1) closing this text box, clicking on üìà Data Collection Information üìâ, then selecting Create Weekly Sheets; or (2) clicking on this hyperlink and giving the browser a few seconds to start the script.
  4. When prompted, enter the date you want to begin data collection, then the number of weekly sheets you would like to create.

Give the script a few seconds to generate the weekly sheets and link them to the Summary sheet. Each weekly sheet includes a column for each weekday, so the dates will begin on the Monday prior to the start date you entered.

Once the script finishes running, you will see the weekly sheets listed across the bottom of the spreadsheet. You are now ready to input data into the Occurrences and Rating cells. The data will automatically update in both the weekly sheets and the summary sheet.
